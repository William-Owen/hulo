hulo: the logger for humans.

Your code has commit messages, capture all the other stuff with hulo.

When working on large or multiple projects, keeping track of what you were thinking when you last looked at the code is almost as important as the latest commits. With hulo you can commit a simple log file along with your code and use the handy command-line tool to update it with the latest meat-space thinking around the project.

Log ig

$ hulo log "I love hulo!"

Read it

$ hulo last

Read the last 3

$ hulo last 3

Count it

$ hulo count coffee 1

Install hulo

Hulo is installed via the NPM package manager.

$ npm i hulo -g
